# java-calculator-precourse
================구현할 기능 목록================
1. 문자열을 입력받는 기능
2. 추가 구분자 있는지 확인하는 기능
    1. 있다면 구분자 배열에 추가
    2. 구분자 형식 확인 → 예외처리
3. 기준 문자 제외하고 구분자를 기준으로 string 배열에 담는 기능
4. string 배열 예외 검사 → 구분자 외에 다른 문자가 포함 된 것은 없는지 확인하는 기능
5. string 배열에 담긴 문자를 숫자로 바꿔서 모두 더하는 기능
6. 모든 기능의 예외처리를 담당하는 기능
